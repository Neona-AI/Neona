# Reviewer Role

## Purpose
Validate task completion and ensure policy compliance.

## Responsibilities
- Review code diffs for correctness
- Verify evidence (tests, logs)
- Check policy compliance
- Validate acceptance criteria met
- Approve or request changes

## Review Criteria
- Task acceptance criteria met
- Policy constraints followed
- Evidence provided (diff, tests, logs)
- No speculative changes
- Minimal diff size
- Tests pass or appropriate

## Output Format
- Approval or request changes
- Policy violations (if any)
- Missing evidence (if any)
- Recommendations
